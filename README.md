# 📊 Dashboard de Produção — Power BI

## 🚀 Sobre o Projeto

Dashboard desenvolvido no Power BI com foco em análise de indicadores de produção e performance operacional.

O objetivo do projeto é transformar dados operacionais em informações estratégicas para apoiar a tomada de decisão.

Durante o desenvolvimento foram realizadas etapas de:
- importação e tratamento de dados no Power BI
- criação de KPIs com medidas DAX
- modelagem de dados
- construção de visualizações interativas
- aplicação de design com foco em usabilidade (Dark Mode)

---

## 🛠️ Ferramentas Utilizadas

- Power BI
- Power Query
- Excel

---

## 📈 Indicadores Analisados

✅ Eficiência operacional  
✅ Disponibilidade de produção  
✅ Produtos aprovados vs rejeitados  
✅ Horas produtivas vs horas paradas  
✅ Performance geral de produção
